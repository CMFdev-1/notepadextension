CMFdev/RoCord Notepad Browser Extension

How to save text
Type the text you'd like to save into the textbox. Then click the "Save text" button.

How to retreive saved text
Click the "Load previously-saved text" button





The saved text will survive the following
  • Browser refreshed
  • Device shutdowns
  • Device hibernations


    • The text will be saved until it is overwritten with other saved text OR the browser data is cleared
